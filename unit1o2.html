Visual Studio Code and Trello are examples of the tools that I used to make my website. Their features helped me in regards to planning and creating it.
In regards to planning, Trello was an influential tool used by me to jot down the different steps I would take to make the website.
Once I finished planning it I started creating the website in Visual Studio Code as it allowed me to implement the three different programming languages which were JavaScript, HTML and CSS.

However if I did not use Visual Studio Code I could have used CodePen. An advantage of using CodePen is that it is an online code editor.
You could write tiny little bits of HTML, CSS and JavaScript code and have immediate visual feedback.
A slight negative is that Code pen does not have an auto-complete system and it does not format on save. People tend to use visual studio code to do their projects and then they paste into CodePen to get a quick url.

Overall, I would say I was happy using Visual Studio Code and Trello as tools in constructing my website.
